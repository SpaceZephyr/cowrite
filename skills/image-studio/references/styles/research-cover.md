# research-cover — 研究报告封面（宽屏横版）

The Economist / a16z research / Stratechery 风格的横版封面，左标题右图标卡片，理性专业。

**专为 2.35:1 / 21:9 横版封面设计**，公众号头图、长文封面、播客 banner 通用。

## Aspect Ratio
- 默认 **21:9**（约等于电影 2.35:1）
- 兼容 16:9，但效果不如宽屏震撼

## Background
- 浅米白 #FAF8F2（主底）
- 可选深蓝 #1E3A8A 大色块占左侧 1/3 形成左右分栏

## Typography
- 主标题：思源宋体 Heavy / Source Serif Pro Black，超大号，深蓝 #1E3A8A
- 副标题：IBM Plex Sans Medium，中性灰 #4B5563
- 卡片小字：IBM Plex Sans Regular，#1F2937
- 标签数字：IBM Plex Mono Bold

## Layout（核心特征）
**左右分栏**，左 55% 文字区，右 45% 图标卡片区：

- **左侧**：
  - 顶部小字标签（如 "AI · 实战手册"），全大写或编号
  - 中部超大主标题（深蓝、占据视觉焦点）
  - 下方一行副标题（灰色，简短）
  - 底部一条细横线 + 作者/日期

- **右侧**：
  - 2–4 个并列信息卡片图标（横向或网格）
  - 每个卡片：单色线条/几何图标 + 一行小字标题 + 编号 ①②③
  - 卡片间留白充足，不堆叠

## Visual Elements
- 图标必须是**单色线条**或**简洁几何**，绝不写实
- 细栅格基线对齐
- 整体留白占比 ≥ 30%
- 文字层级清晰：标签 < 副标题 < 主标题
- 配色冷静专业，不用渐变

## Do
- 主标题足够大、震撼
- 图标抽象简洁，承担信息职责
- 严格栅格对齐
- 中文标题用宋体，英文用衬线
- 整体像研究报告 / 行业白皮书封面

## Don't
- 卡通图标 / emoji
- 3D 图表 / 渐变阴影
- 装饰性插画
- 多种主色乱用
- 字体超过 3 种
